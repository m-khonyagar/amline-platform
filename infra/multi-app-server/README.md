# میزبانی چند اپلیکیشن روی یک سرور

این پوشه قراردادها و چک‌لیست‌ها را برای دیپلوی چند پروژه روی **یک میزبان لینوکس** (مثل `212.80.24.109`) نگه می‌دارد تا مسیرها، پورت‌ها و نام سرویس‌های systemd با هم قاطی نشوند.

| فایل | محتوا |
|------|--------|
| [LAYOUT.md](./LAYOUT.md) | ساختار `/opt/apps/` و نام‌گذاری |
| [PORT-REGISTRY.md](./PORT-REGISTRY.md) | جدول پورت‌ها (با IP ثبت‌شده) |
| [NEW-APPLICATION.md](./NEW-APPLICATION.md) | چک‌لیست اپ جدید |

**ثابت‌های پایتون** (مسیر پیش‌فرض دیپلوی املاین staging): `scripts/server_layout_constants.py`

**اسکریپت کامل** (بیلد + استخراج + systemd + مهاجرت از مسیر قدیمی): `scripts/full_staging_provision.py`
