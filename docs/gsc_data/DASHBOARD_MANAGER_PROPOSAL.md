# پیشنهاد داشبورد مدیریتی سئو — املاین

## هدف
نمایش عملکرد سئو به‌صورت حرفه‌ای برای مدیر، با ترکیب داده‌های ارگانیک، تبلیغات و تبدیل (conversion).

---

## ۱. شاخص‌های پیشنهادی برای اضافه شدن

### الف) تبلیغات و کمپین
| شاخص | توضیح | منبع داده |
|------|--------|-----------|
| **هزینه کمپین** | مجموع هزینه تبلیغات در بازه | گوگل ادز، تبلیغات شبکه‌های اجتماعی، دستی |
| **تاریخ کمپین** | شروع و پایان هر کمپین | دستی |
| **کلیک کمپین** | کلیک از تبلیغات | گوگل ادز / Analytics |
| **نمایش کمپین** | ایمپرشن تبلیغات | گوگل ادز |
| **هزینه به ازای کلیک (CPC)** | هزینه ÷ کلیک | محاسبه |
| **هزینه به ازای نمایش (CPM)** | هزینه ÷ (نمایش/۱۰۰۰) | محاسبه |

### ب) ورود کاربر و attribution
| شاخص | توضیح | منبع داده |
|------|--------|-----------|
| **لاگین از کمپین** | تعداد کاربری که از لینک کمپین لاگین کرده‌اند | بک‌اند / Analytics |
| **لاگین از سرچ ارگانیک** | تعداد کاربری که از گوگل ارگانیک آمده‌اند | Analytics / UTM |
| **ورود مستقیم** | کاربرانی که مستقیم (URL، بوکمارک) آمده‌اند | Analytics |
| **درصد ارگانیک** | لاگین ارگانیک ÷ کل لاگین | محاسبه |
| **درصد کمپین** | لاگین کمپین ÷ کل لاگین | محاسبه |
| **درصد مستقیم** | ورود مستقیم ÷ کل | محاسبه |

### ج) تبدیل و خرید
| شاخص | توضیح | منبع داده |
|------|--------|-----------|
| **تعداد خرید** | از ارگانیک / کمپین / مستقیم | بک‌اند / Analytics |
| **درآمد** | فروش ناشی از هر منبع | بک‌اند |
| **هزینه به ازای خرید (CPA)** | هزینه کمپین ÷ تعداد خرید از کمپین | محاسبه |
| **ROAS** | درآمد ÷ هزینه تبلیغات | محاسبه |
| **ارزش کلیک ارگانیک** | (درآمد ارگانیک ÷ کلیک ارگانیک) | محاسبه |

### د) مقایسه ارگانیک vs تبلیغات
| شاخص | توضیح |
|------|--------|
| **نسبت کلیک ارگانیک به کمپین** | کلیک GSC ÷ کلیک کمپین |
| **کارایی هزینه** | هزینه تبلیغات برای جذب X کاربر vs ارزش X کلیک ارگانیک |
| **تطبیق تاریخ** | هم‌راستایی کمپین با روند کلیک GSC در همان بازه |

## ۲. ساختار شیت‌های پیشنهادی

### شیت «کمپین_تبلیغاتی»
| ستون | نوع | توضیح |
|------|-----|--------|
| تاریخ_شروع | تاریخ | شروع کمپین |
| تاریخ_پایان | تاریخ | پایان کمپین |
| نام_کمپین | متن | نام کمپین |
| هزینه | عدد | هزینه (تومان یا دلار) |
| کلیک | عدد | کلیک از تبلیغ |
| نمایش | عدد | ایمپرشن تبلیغ |
| لاگین | عدد | کاربران لاگین‌شده از این کمپین |
| خرید | عدد | تعداد خرید از کمپین |
| درآمد | عدد | درآمد ناشی از کمپین |

### شیت «ورود_کاربران_روزانه»
| ستون | نوع | توضیح |
|------|-----|--------|
| تاریخ | تاریخ | روز |
| ارگانیک | عدد | لاگین/ورود از سرچ |
| کمپین | عدد | لاگین از تبلیغات |
| مستقیم | عدد | ورود مستقیم |
| سایر | عدد | رفرال، شبکه‌های اجتماعی و غیره |

### شیت «خلاصه_مدیر»
ترکیب خودکار از GSC + کمپین + ورود کاربران با نمودار و KPIهای مدیریتی.

---

## ۳. منبع داده و نحوه پر شدن

| داده | منبع پیشنهادی | روش دریافت |
|------|----------------|------------|
| GSC | Google Search Console API | ✅ خودکار (فعلی) |
| تبلیغات | Google Ads API / گزارش دستی | API یا ورود دستی در شیت |
| لاگین/منبع | Google Analytics 4 / بک‌اند | GA4 API یا export |
| خرید/درآمد | بک‌اند املاین | API داخلی یا export |

---

## ۴. فرمول‌های محاسباتی (در گوگل شیت)

```
CPC = هزینه / کلیک_کمپین
CPM = (هزینه / نمایش_کمپین) * 1000
CPA = هزینه / خرید_از_کمپین
ROAS = درآمد_کمپین / هزینه_کمپین
درصد_ارگانیک = لاگین_ارگانیک / کل_لاگین
ارزش_کلیک_ارگانیک = درآمد_ارگانیک / کلیک_GSC
```

---

## ۵. نمودارهای پیشنهادی برای مدیر

1. **روند کلیک ارگانیک vs کمپین** (خطی دو سری)
2. **توزیع منبع ورود** (دایره‌ای: ارگانیک، کمپین، مستقیم)
3. **هزینه vs درآمد** (ستونی)
4. **ROAS در زمان** (خطی)
5. **مقایسه CPC ارگانیک (ارزش کلیک) با CPC کمپین**
