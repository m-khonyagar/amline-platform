# فلو پیش‌فروش آپارتمان (PRE_SALE)

SSOT kind: `PRE_SALE` — نوع API: `PRE_SALE`.

## مراحل

1. `POST /api/v1/contracts/start` با `contract_type: PRE_SALE`.
2. نقش‌ها: `DEVELOPER` / `BUYER`.
3. `terms`: نام پروژه، شماره واحد، قیمت کل، `payment_schedule` مراحل قانونی، تاریخ تحویل، جریمه تأخیر.

## چک‌لیست انطباق (محصول)

- [ ] تطبیق با قانون پیش‌فروش ساختمان (نسخهٔ معتبر زمان اجرا)
- [ ] مستندات ضمیمه الزام‌آور (پروانه، بیمه، حساب امانی در صورت نیاز)

جزئیات حقوقی در سند جداگانه تیم حقوقی نگهداری شود.
