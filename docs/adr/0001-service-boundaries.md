# ADR 0001 — مرز سرویس‌های بک‌اند (پیش‌نویس)

## وضعیت

پیش‌نویس — 2026-04

## زمینه

مونولیت FastAPI در `backend/backend` شامل auth، قرارداد، کیف پول، نوتیفیکیشن و غیره است.

## تصمیم (هدف)

1. **Auth** اولین نامزد استخراج به سرویس جدا با همان قرارداد `/api/v1` یا gateway.
2. **Notifications** به صف رویداد (NATS / Redis Stream) و worker جدا متکی شود.
3. **Wallet** با ledger داخلی و API sync با درگاه؛ DB جدا یا schema جدا قبل از سرویس مستقل.

## پیامدها

- نیاز به **correlation id** و **service token** بین سرویس‌ها.
- مهاجرت تدریجی: ماژول‌های پایتون با مرز `app/services/*` سپس بستهٔ deploy جدا.

## مراحل بعد

- تعریف رویدادهای دامنه در `docs/event-driven.md`.
- PR جدا: اسکلت سرویس نوتیفیکیشن.
