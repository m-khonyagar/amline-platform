## جدول ماتریسی ارزیابی

| پارامتر بررسی‌شده | وضعیت فعلی | ارزیابی عملکرد سئوکار | پیشنهاد اجرایی |
|-------------------|------------|------------------------|-----------------|
| **۱. روند ترافیک (Clicks)** | *درج میانگین کلیک‌ها و روند ماهانه* | عالی / نیاز به بهبود / بحرانی | *آیا روند صعودی است؟ آیا با فصل همخوان است؟* |
| **۲. روند ایمپرشن (Impressions)** | *درج میانگین ایمپرشن و روند* | عالی / نیاز به بهبود / بحرانی | *مقایسه با دوره قبل؛ تشخیص سیزنال vs بهبود واقعی* |
| **۳. کلمات کلیدی هدف (Targeted Keywords)** | *لیست کلمات + میانگین رتبه* | عالی / نیاز به بهبود / بحرانی | *صفحه اول (۱–۱۰) یا نه؛ پیشنهاد بهینه‌سازی* |
| **۴. کلمات با افت رتبه (Average Position ↓)** | *کلمات با بیشترین افت + مقدار افت* | عالی / نیاز به بهبود / بحرانی | *دلایل احتمالی؛ اقدام اصلاحی* |
| **۵. نرخ کلیک (CTR) نسبت به رتبه** | *CTR واقعی vs CTR پیش‌فرض گوگل برای هر رتبه* | عالی / نیاز به بهبود / بحرانی | *بهبود تایتل و متا دیسکریپشن؛ تست A/B* |
| **۶. ایندکس و Coverage** | *تعداد صفحات ایندکس‌شده؛ خطاهای جدید* | عالی / نیاز به بهبود / بحرانی | *رفع 404، سرور ارور، و صفحات حذف‌شده* |
| **۷. Core Web Vitals (موبایل)** | *LCP, FID/INP, CLS — وضعیت Good/Need Improvement/Poor* | عالی / نیاز به بهبود / بحرانی | *بهینه تصاویر، JS، layout shift* |
| **۸. Core Web Vitals (دسکتاپ)** | *همان متریک‌ها برای دسکتاپ* | عالی / نیاز به بهبود / بحرانی | *مقایسه با موبایل؛ اولویت رفع* |

## خلاصه تحلیل به تفکیک بخش

### ۱. تحلیل روند ترافیک و کلیک‌ها
- **روند:** صعودی / نزولی / ثابت  
- **نتیجه:** تغییرات ناشی از سیزنال بودن است / بهبود عملکرد سایت است / ترکیبی  
- **داده‌های کلیدی:** *Clicks و Impressions در بازه موردنظر*

### ۲. تحلیل کلمات کلیدی
- **کلمات با افت رتبه:** *لیست + مقدار افت*
- **کلمات هدف در صفحه اول:** بله / خیر — *شرح*

### ۳. نرخ کلیک (CTR)
- **منطقی بودن CTR با توجه به رتبه:** بله / خیر  
- **پیشنهاد برای تایتل و دیسکریپشن:** *اقدامات مشخص*

### ۴. ایندکس و خطاها
- **خطاهای جدید (۴۰۴، سرور، و غیره):** دارد / ندارد — *شرح*
- **وظیفه سئوکار:** رفع شده / نیاز به رفع

### ۵. Core Web Vitals
- **موبایل:** *وضعیت*
- **دسکتاپ:** *وضعیت*
- **افت در تجربه کاربری:** دارد / ندارد

## نحوه استفاده از این قالب

برای پر کردن این گزارش با **داده واقعی**، لطفاً یکی از موارد زیر را ارائه دهید:

1. **فایل خروجی گوگل سرچ کنسول:**  
   - Performance (Query / Page / Country) به صورت CSV  
   - Indexing > Pages (اگر خطا دارید)  
   - Experience > Core Web Vitals (خلاصه یا اسکرین)

2. **جدول یا متن خلاصه:**  
   - میانگین Clicks و Impressions در ۳–۶ ماه اخیر  
   - لیست کلمات کلیدی مهم + Average Position و CTR  
   - هر خطای Coverage/Indexing که مشاهده شده  
   - وضعیت CWV (Good/Need Improvement/Poor) برای موبایل و دسکتاپ  

پس از دریافت داده‌ها، این سند با ارزیابی واقعی (عالی/نیاز به بهبود/بحرانی) و پیشنهادات اجرایی به‌روزرسانی می‌شود.
