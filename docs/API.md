# API Documentation

API اصلی در `packages/api` پیاده‌سازی شده و endpointها در namespace زیر ارائه می‌شوند:

- `/api/auth`
- `/api/properties`
- `/api/payments`
- `/api/billing`
- `/api/licenses`
- `/api/achievements`
- `/api/hr`
- `/api/admin`

## Design Principles

- versionable routes
- structured error responses
- request logging
- rate limiting
- service layer isolation
