# Monorepo (اختیاری)

برای اشتراک **Design System** و **api-client** بین `admin-ui` و `amline-ui`:

- ساختار پیشنهادی: `packages/ui`, `packages/api-client` با pnpm workspaces یا Turborepo.
- تولید client از OpenAPI رسمی (`amline-dev`) برای یک منبع تایپ واحد.
- این مخزن فعلاً چند پروژهٔ جداست؛ این سند فقط نقشهٔ راه است تا با تصمیم تیم اجرا شود.
