راهنمای تست محلی — نقش کاربر عادی (seller)

این فقط برای localhost است. نشست آزمایشی بدون توکن واقعی است؛ برای OTP واقعی API را روی پورت ۸۰۸۰ اجرا کنید و از صفحه ورود استفاده کنید.

۱) اعمال سریع نشست کاربر (بدون OTP)

۲) صفحات پس از ورود (کاربر عادی)

  • نشان‌شده‌ها

  • تاریخچه پرداخت
  • قراردادها
  • قرارداد جدید
  • جزئیات نمونه ct-1001
  • جزئیات نمونه ct-1002
  • گفتگو

  • گفتگو need-250
  • پشتیبانی
