ارزش و فیچرهای پلتفرم (خلاصهٔ فنی–کسب‌وکار)

انعقاد و مدیریت قراردادویزارد چندمرحله‌ای، امضای OTP، شاهد، PDF؛ همسو با dev-mock-api و بک‌اند FastAPI.
چند وجهی (ادمین / کاربر / مشاور)سه رابط جدا با مجوزهای متفاوت؛ یک قرارداد و API مشترک.
CRM و لیدکانبان، فعالیت‌ها، اتصال به ماژول‌های ادمین طبق مسیرهای /admin/crm.
مالی و کیف پولمسیرهای مالی و ابزارهای Hamgit در پنل ادمین (بخش‌های توسعه‌یافته).
حاکمیت و امنیتنقش‌ها، ممیزی، گزارش فعالیت؛ آمادهٔ اتصال به production پس از پیکربندی.
سایت و سئوسایت استاتیک و داشبورد سئو برای کانال بازاریابی جدا از اپ اصلی.

پنل ادمین

  • داشبورد KPI، قراردادها، ویزارد، کاربران، CRM، کیف پول، نقش‌ها، ممیزی
  • ورود آزمایشی در dev با تنظیم env (ر. docs/LOCAL_DEV.md)

پنل کاربر نهایی (amline-ui)

  • قراردادها، نیازمندی، بازار، کیف پول؛ مسیر استاندارد OTP

پنل مشاور (consultant-ui)

  • ورود مشاور؛ اتصال به /consultant/* روی mock یا بک‌اند

برای بالا آوردن API و باز خودکار این صفحه در ویندوز: powershell -ExecutionPolicy Bypass -File .\scripts\start-platform-local-hub.ps1 -WithUserUi -WithAdminUi -WithConsultantUi — فقط هاب + mock: همان اسکریپت را بدون سوییچ اجرا کنید؛ سپس دستی npm run dev در هر پوشه.
